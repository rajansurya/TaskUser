FAQ

  1. Why task/task message created by me is not visible to assignee.

    Task/task message will not be visible to assignee in following scenarios:

  1. . Task is still lying within your application as you are not internet connected. Connect to internet and within 30sec it goes out

  2. . Assignee is not internet connected. Connect to internet and within 30sec shall receive it.

  3. . Contact chosen as assignee is having only email id/facebook/linkedin Id etc but not mobile number. Select the correct contact with mobile number



  1. What will happen to assigned task if assignee is not having the assigner application.

    Assignee in this case will receive SMS of the task along with a link to download this application.SMS will be sent via you mobile phone only so the SMS charges as per your plan will apply



  2. What charges shall be applicable if I send tasks to my colleagues who are internationally located?

    Application works on internet so no charges will apply if assignee has assigner application to receive it. Otherwise SMS will be sent and charges as per your plan will apply.

  3. Why I am unable to change urgency level to fire for few tasks?

    These tasks are assigned by someone who can only modify urgency level. Use “closely watched” feature to make it a high priority item for you.


  4. Why I am unable to edit or redefine deadline or reassign the task in few cases?

    These tasks are assigned by someone who as owner can only modify it


  5. Task marked as closely watched by me will be visible as closely watched to the assignee or assignor also?

    No, closely watched is for only your prioritization so view of others will not be impacted if you mark any tasks as closely watched.


  6. From where can I access closed tasks ?

    From menu options

  7. Sync happens at what frequency?


At every 1 minute but can use sync option from menu for instant one

  1. From where I can view all the tasks sent by someone
    Use “sort by name” option from menu

Use “contact wise summary” option from menu
Though I chose priority level while creating a task but where is it being used except for the tasks with urgency as fire?

Use “sort by priority” option from menu to view task as per priority

  1. How to send reminder to the assignee of task

Use messaging option to sent reminder and a request to expedite

Task with a new message comes on top, this way is more prominently visible to assignee.

  1. By using this application someone can bombard me with tasks and choke my working area
    We have put a protection that messages from unknown mobile number (i.e. from a mobile number which is not in contact list ) go to junk section.

Tips to use it effectively

  1. Create a task and assign it an assignee as when you have realization of some work to be done by someone else.

  2. Use deadline only for the tasks which actually has a deadline.

  3. Maintain tasks list not more than 50 , if it is more, then it is the time to introspect.

  4. Do not use fire indiscreetly otherwise it shall lose its importance.

  5. Use “closely watched” options if you have too many important tasks to manage but you can focus only on few.

  6. To send a reminder use “send message” feature as this will not only send a reminder but will also push the task on the top in summary.
